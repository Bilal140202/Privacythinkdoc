# Getting Started with PrivacyThink

Welcome! This guide will help you get the PrivacyThink landing page up and running.

### 1. Clone and Install

```bash

```

### 2. Set Up Convex Backend

```bash

# - Start the dev server
```

## ğŸŒ Deploy to Netlify (10 minutes)

### Step 1: Push to GitHub

```bash
# Initialize git (if not already done)
git init
git add .
git commit -m "Initial commit"

# Push to GitHub
git branch -M main
git remote add origin https://github.com/yourusername/privacythink.git
git push -u origin main
```

### Step 2: Connect to Netlify

1. Go to [Netlify](https://app.netlify.com)
2. Click "Add new site" â†’ "Import an existing project"
3. Select your GitHub repository
4. Settings are auto-detected from `netlify.toml`
5. Click "Deploy"

### Step 3: Add Environment Variables

1. Go to Site settings â†’ Environment variables
2. Add:
   - `VITE_CONVEX_URL` (from your .env.local)
   - `CONVEX_DEPLOYMENT` (from your .env.local)
3. Trigger a redeploy

### Step 4: Custom Domain (Optional)

1. Site settings â†’ Domain management
2. Add your custom domain
3. Configure DNS records with your provider
4. Wait for DNS propagation (5-30 minutes)

**Your site is now live!** ğŸ‰

## ğŸ”§ Common Tasks

### Add a New Page

1. Create component in `src/pages/NewPage.tsx`:
   ```tsx
   export default function NewPage() {
     return <div>My New Page</div>;
   }
   ```

2. Add route in `src/main.tsx`:
   ```tsx
   const NewPage = lazy(() => import("./pages/NewPage.tsx"));

   // In Routes:
   <Route path="/newpage" element={<NewPage />} />
   ```

3. Add link in navigation/footer

### Update Colors

Edit `src/index.css`:

```css
:root {
  --primary: oklch(0.55 0.25 285);    /* Purple */
  --secondary: oklch(0.6 0.2 265);     /* Blue */
  --accent: oklch(0.65 0.25 330);      /* Pink */
}
```

### Add a Shadcn Component

```bash
# Example: Add a table component
npx shadcn-ui@latest add table
```

Components are added to `src/components/ui/`

## ğŸ› Troubleshooting

### Port Already in Use

```bash
# Kill process on port 5173
npx kill-port 5173

# Or use a different port
pnpm dev -- --port 3000
```

### Module Not Found Errors

```bash
# Clear cache and reinstall
rm -rf node_modules .pnpm-store
pnpm install
```

### Build Fails

```bash
# Check for TypeScript errors
npx tsc -b --noEmit

# Check for linting issues
pnpm lint
```

### Convex Connection Issues

```bash
# Restart Convex dev server
npx convex dev --once  # Run once
npx convex dev         # Watch mode
```
