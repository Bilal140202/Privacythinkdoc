## How to Update

### Desktop Application
PrivacyThink will automatically check for updates and notify you when a new version is available. You can also manually check in Settings > About > Check for Updates.

### Landing Page
The landing page is continuously deployed via Netlify. Changes are live immediately after pushing to the main branch.
